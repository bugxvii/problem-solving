## 문제
- [1008번 A\\B](https://www.acmicpc.net/problem/1008)

## C
```cpp
#include <stdio.h>

int main(void)
{
	double a, b;
	scanf("%lf %lf", &a, &b);
	printf("%.10g\n", a/b);
	return 0;
}
```

## Time Complexity
- 시간 복잡도: <b>O(1)</b>.

## Space Complexity
- 공간 복잡도: <b>O(1)</b>.

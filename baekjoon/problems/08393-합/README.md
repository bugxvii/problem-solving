## 문제
- [8393번 합](https://www.acmicpc.net/problem/8393)

## C
```cpp
#include <stdio.h>

int main(void)
{
	int n;
	scanf("%d", &n);
	printf("%d\n", (n*(n+1))/2);
	return 0;
}
```

## Time Complexity
- 시간 복잡도: <b>O(1)</b>.

## Space Complexity
- 공간 복잡도: <b>O(1)</b>.

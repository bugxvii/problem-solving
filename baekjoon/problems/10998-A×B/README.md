## 문제
- [10998번 A\*B](https://www.acmicpc.net/problem/10998)

## C
```cpp
#include <stdio.h>

int main(int argc, char **argv)
{
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%d\n", a*b);

	return 0;
}
```

## Time Complexity
- 시간 복잡도: <b>O(1)</b>.

## Space Complexity
- 공간 복잡도: <b>O(1)</b>.
